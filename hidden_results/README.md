# hidden_results/

Optional folder for pre-generated JSON / binary result files that should
be tracked in the git history despite being too large for the standard
`results/` directory (which is gitignored and regenerated by `make all`).

## Usage

If any result JSON exceeds ~500 KB or you want a specific snapshot
permanently available without regenerating everything, move it here and
git-add it:

```bash
mv results/some_large_file.json hidden_results/
git add hidden_results/some_large_file.json
git commit -m "Add large result file to hidden_results"
```

The `hidden_results/` directory itself is tracked by git but is excluded
from `.gitignore` (unlike `results/`).  Reviewers cloning the repo should
read these JSONs directly; developers should regenerate them via
`make all` (Unix) or `.\run_all.ps1` (Windows).
